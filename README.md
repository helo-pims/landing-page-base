# landing-page-base
Template escalável e responsivo para Landing Pages, focado em captura de leads e arquitetura ágil.

## Base Landing Page

> Uma estrutura de landing page minimalista, desenvolvida com foco em conversão direta e atrito zero.

## O Propósito

Este repositório contém uma base técnica escalável. Ele foi projetado para atuar como uma **ferramenta de alinhamento visual e estratégico**. 

É uma forma prática e interativa de fazer com que **clientes não técnicos entendam a anatomia de uma landing page**. Através deste modelo, é possível visualizar na prática como os blocos de informação (Header, Sobre, Serviços e CTAs) se conectam de forma lógica para guiar o usuário final até a ação de conversão (contato via WhatsApp).

## Tecnologias Utilizadas

A stack foi escolhida para garantir carregamento instantâneo, facilidade de manutenção e uma experiência mobile impecável:

* **[React](https://reactjs.org/)** + **[Vite](https://vitejs.dev/)**: Para um ambiente de desenvolvimento ultra-rápido e build otimizado.
* **[Tailwind CSS](https://tailwindcss.com/)**: Estilização utilitária focada em design clean, minimalista e sem excessos (zero gradientes, paleta sóbria).
* **[Framer Motion](https://www.framer.com/motion/)**: Para micro-interações e animações sutis que agregam valor sem pesar a interface.
* **[Lucide React](https://lucide.dev/)**: Ícones leves e consistentes.

## Funcionalidades Principais

* **Mobile-First Absoluto**: Layout testado e fluido para a melhor experiência em dispositivos móveis.
* **Roteamento Direto (WhatsApp)**: CTAs configurados com links dinâmicos e mensagens pré-definidas para segmentar a entrada do lead.
* **Design System Limpo**: Uso de tons de cinza com acentos pontuais (azul/roxo) apenas para interações de *hover* e *micro-feedbacks*.
* **Zero Ruído**: Remoção de elementos visuais desnecessários para focar 100% na atenção do usuário.
